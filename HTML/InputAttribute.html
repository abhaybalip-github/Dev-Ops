This is a form
Form elements are placed here

The HTML input element is the most used form element.
An input element can be displayed in many ways, depending on the type attribute.
types of Attribute :

value attribute - place value data in textbox


readonly attribute - user can only read value data


disabled attribute - textbox is disabled


size attribute - define size of textbox


maxlength attribute - define maximum length for text data


min max attribute - define minimum and maximum data for input


multiple attribute - allow user to enter more than one data


placeholder attribute - given data takes place until user enters


pattern attribute - input of a specific pattern/ used for specific input matching


required attribute - defines a data necessary to input


step attribute - input can be only multiple of given number


autofocus attribute - focuses on this input field on opening


autocomplete attribute - allow user to insert data from autocomplete dict
